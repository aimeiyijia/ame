## 测试核心
### 单元测试
选定单元测试框架，创建测试规则，指定测试覆盖率最小值
### 组件测试
功能测试，行为测试
### 接口测试
后端验证 API 是否与 Mock 数据统一，前端验证 Mock 数据或 API 是否满足前端需求
### E2E 测试
模仿用户行为，测试参数、参数类型、参数值、参数数量、返回值、抛出错误等，保证系统能够在任何情况下都稳定可靠完成工作
